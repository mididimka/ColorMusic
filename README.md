# ColorMusic

ColorMusic — это WPF-приложение на C# для визуализации музыки в виде 8 цветовых прожекторов, реагирующих на аудио, которое воспроизводится на компьютере.

Программа захватывает звук с выхода звуковой карты через loopback, анализирует спектр и распределяет реакцию по частотным диапазонам. Каждый канал можно настроить отдельно: цвет, диапазон частот, чувствительность, форма, размер и положение на экране.

## Возможности

- 8 независимых прожекторов
- захват системного звука с выхода звуковой карты
- отдельные диапазоны частот для каждого канала
- настройка цвета каждого канала
- выбор формы прожектора:
  - круг
  - квадрат
  - треугольник
- регулировка:
  - общего уровня
  - общей чувствительности
  - прозрачности прожекторов
  - цели автоуровня
  - чувствительности каждого канала
  - усиления каждого канала
  - размера каждого канала
- сохранение настроек в `settings.json`
- автоматическая загрузка настроек при запуске
- окно спектра по каналам
- полноэкранный режим
- фоновое звёздное небо
- отключаемый дым
- отключаемые эффекты:
  - автоуровень
  - пульсация
  - стробоскоп
  - вспышки по битам
  - неоновое свечение
- регулировка силы эффектов:
  - пульсация
  - стробоскоп
  - бит-вспышки
  - неон

Как это работает
Программа получает системный звук через WasapiLoopbackCapture.
Аудиопоток преобразуется в набор сэмплов.
Выполняется FFT-анализ спектра.
Для каждого канала вычисляется амплитуда в заданном частотном диапазоне.
Полученные значения проходят через чувствительность, усиление, автоуровень и визуальные эффекты.
Прожекторы обновляют яркость и свечение в реальном времени.
Управление
F1 — показать или скрыть панель настроек
F2 — показать или скрыть окно спектра
F11 — полноэкранный режим
Настройки

Все настройки сохраняются в файл:

settings.json

При следующем запуске программа автоматически подгружает сохранённые параметры.

Примечания
Для корректной работы приложение должно получать звук с устройства вывода Windows по умолчанию.
Если спектр или прожекторы реагируют слишком слабо или слишком сильно, отрегулируйте:

Ур.

неон
